# 02. 화면 설계 (UX/UI)

## 2.1 전체 화면 목록

### Onboarding Flow (최초 1회)
| ID | 화면명 | 목적 |
|----|--------|------|
| OB-01 | WelcomeScreen | 앱 소개 및 진입 |
| OB-02 | ClassSelectionScreen | 12 클래스 선택 |
| OB-03 | AbilityRollScreen | 능력치 굴림 |
| OB-04 | CharacterNamingScreen | 캐릭터 이름/아바타 |
| OB-05 | GTDTutorialScreen | GTD 5단계 튜토리얼 |

### Main Navigation (Bottom Bar 5탭)
| ID | 화면명 | GTD 단계 | 탭 아이콘 |
|----|--------|----------|----------|
| M-01 | InboxScreen | Capture | 📥 |
| M-02 | QuestBoardScreen | Organize + Engage | ⚔️ |
| M-03 | CombatScreen | 전투 현황 | 🔥 |
| M-04 | JournalScreen | Reflect | 📖 |
| M-05 | CharacterSheetScreen | 캐릭터 관리 | 🧙 |

### Detail Screens (Stack Navigation)
| ID | 화면명 | 진입 경로 |
|----|--------|----------|
| D-01 | TaskDetailScreen | QuestCard 탭 |
| D-02 | ProjectDetailScreen | 프로젝트 카드 탭 |
| D-03 | CombatResultScreen | 퀘스트 완료 후 |
| D-04 | LevelUpScreen | XP 임계값 도달 시 |
| D-05 | ItemDetailScreen | 아이템 탭 |
| D-06 | NpcScreen | CharacterSheet → NPC 탭 |
| D-07 | WeeklyReviewScreen | Journal → 주간 리뷰 탭 |
| D-08 | EncounterLogScreen | Combat → 인카운터 탭 |
| D-09 | AchievementScreen | CharacterSheet → 업적 탭 |
| D-10 | SettingsScreen | TopAppBar → 설정 |
| D-11 | ClassDetailScreen | ClassSelection 또는 Settings |
| D-12 | StatisticsScreen | Journal → 통계 탭 |

### Modal / Bottom Sheet
| ID | 컴포넌트명 | 트리거 |
|----|-----------|--------|
| BS-01 | QuickCaptureSheet | 모든 화면 FAB |
| BS-02 | ClarifySheet | Inbox 스와이프 또는 탭 |
| BS-03 | D20RollSheet | 퀘스트 완료 체크 |
| BS-04 | ClassCompatibilitySheet | NPC 궁합 보기 |
| BS-05 | ContextFilterSheet | QuestBoard 필터 탭 |
| BS-06 | DatePickerSheet | 마감일 설정 |
| BS-07 | CrAdjustSheet | CR 수동 조정 |

      │
