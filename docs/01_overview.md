# 01. 앱 개요 및 철학

## 1.1 프로젝트 비전

**QuestLog**는 GTD의 마찰 없는 수집·실행 루프와 D&D의 서사적 보상 루프를 단일 UX 흐름으로 통합한다.

기존 생산성 앱의 문제: 할 일을 완료해도 보상이 없고, 지루하다.
기존 게임의 문제: 현실 생산성과 연결되지 않아 의미가 없다.

**QuestLog의 해법**: 실제 할 일 완료 → 즉각적 게임 보상 → 캐릭터 성장 → 더 강한 동기 유발

앱은 "생산성 도구를 게임화"하는 것이 아니라, **"삶의 모험을 체계화"**하는 프레임을 제공한다.
이는 책 *How to Be More D&D*의 핵심 철학에서 출발한다:

> "What is life if not a series of random encounters?"

## 1.2 핵심 철학 (Design Principles)

### 1) Zero Friction Capture (마찰 없는 수집)
- 플로팅 버튼은 모든 화면에서 항상 접근 가능
- 텍스트 한 줄 → 즉시 저장 (Clarify는 나중에)
- 음성 입력 지원
- 앱을 열지 않아도 빠른 수집 가능 (알림 위젯)

### 2) Instant Reward Loop (즉각적 보상)
- 완료 체크 → D20 굴림 → 결과 → XP 획득 전체가 **3초 이내** 완결
- 보상이 지연되면 습관화가 깨진다
- 크리티컬 히트·미스도 유머러스하게 처리하여 실패에 대한 부담 제거

### 3) Yes, And... Philosophy (긍정과 확장)
- 책의 "Yes, and..." 임프로비제이션 철학 적용
- 앱은 절대 "안 된다"고 하지 않는다
- 완벽하지 않아도 기록되고, 기록되면 XP가 있다
- 마감을 넘긴 퀘스트도 완료 시 보상 (단, 페널티 포함)

### 4) DM of Your Own Life (삶의 던전마스터)
- Weekly Review = 자신의 삶을 DM 시점에서 조망하는 의식적 세션
- 메트릭보다 내러티브를 강조
- Claude AI가 영웅의 일지를 작성해준다

### 5) Fair Dice (공정한 주사위)
- D20는 진짜 난수 (SecureRandom)
- 조작 없음. 불운도 스토리의 일부
- 단, 클래스·장비·스트릭으로 확률을 개선할 수 있다

## 1.3 타겟 사용자

### Primary Target
- **나이**: 25-40세
- **특성**: GTD 또는 생산성 방법론에 관심 있는 직장인/프리랜서
- **게임 경험**: D&D 또는 RPG에 친숙하거나 관심 있음
- **페인포인트**: 할 일 앱을 꾸준히 사용하지 못하고 이탈
- **니즈**: 생산성 유지의 동기 부여 메커니즘

### Secondary Target
- GTD 초보자이지만 게임을 통해 방법론을 자연스럽게 학습하고 싶은 사람
- D&D 팬이지만 실제 삶에 적용하고 싶은 사람

### Anti-Target (이 앱이 맞지 않는 사람)
- 단순한 체크리스트 앱을 원하는 사람
- 게임 요소를 싫어하는 사람
- 즉각적인 복잡한 협업 도구가 필요한 팀

## 1.4 경쟁 앱 분석

| 앱 | 강점 | 약점 | QuestLog 차별점 |
|----|------|------|-----------------|
| Habitica | 게임화 잘 됨 | GTD 구조 없음, 비주얼 단조 | GTD 5단계 완전 통합 |
| Todoist | GTD 지원 탁월 | 게임 요소 전무, 지루함 | D&D 세계관 + 전투 시스템 |
| Things 3 | UX 완성도 높음 | iOS 전용, 게임 요소 없음 | 안드로이드 + AI 스토리 |
| Forest | 집중 게임화 | 할 일 관리 없음 | 할 일 완료가 핵심 트리거 |
| D&D Beyond | D&D 완성도 | 현실 연결 전혀 없음 | 현실 GTD와 완전 연결 |

## 1.5 핵심 지표 (Success Metrics)

- **Day 7 리텐션**: 40% 이상 (일반 할 일 앱 평균: 20%)
- **Day 30 리텐션**: 20% 이상
- **주간 리뷰 완료율**: 사용자의 30% 이상이 매주 완료
- **일일 활성 퀘스트 완료 수**: 평균 3개 이상
- **평점**: 플레이스토어 4.2 이상
- **스트릭 7일 달성률**: 신규 사용자의 25% 이상
